# Incident Response

## Immediate actions

- Safe up or dock immediately.
- Inform Dispatch with system name, threat type, and status.
- Do not engage unless explicitly ordered.

## Standard report format

```
INCIDENT REPORT
Pilot: <name>
System: <system>
Threat: <type>
Status: <safe / under fire / lost>
```

## Post-incident review

Dispatch will log the incident and issue follow-up instructions.
